# scripts/test-orchestrator-scope-sync.sh — contract

`scripts/test-orchestrator-scope-sync.sh` is the cross-validation harness (closes #285) that asserts exact set equality between the `ORCHESTRATORS` and `DELEGATORS` bash arrays in `scripts/test-anti-halt-banners.sh` and the bulleted scope lists under `### Scope list` in `skills/shared/subskill-invocation.md` (the "banner MUST appear" orchestrator list and the "banner MUST NOT appear" pure-delegator list). Emits symmetric diff-style drift messages naming entries missing on each side; fail-closed on empty parse. Wired into `make lint` via the `test-orchestrator-scope-sync` target; added to `agent-lint.toml`'s exclude list because agent-lint does not follow Makefile-only references. Edits to either source — the `ORCHESTRATORS`/`DELEGATORS` array in `scripts/test-anti-halt-banners.sh` or the `### Scope list` bullets in `skills/shared/subskill-invocation.md` — must keep the two sides in sync in the same PR; the harness is the mechanical enforcer.
