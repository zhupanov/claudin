Bailing to cleanup.**` bail string appear; (H) exactly one `rebase-push.sh --no-push --skip-if-pushed` occurrence in the entire file — residual inline rebase blocks from the pre-refactor 4-copy layout would push the count to 2+ and fail; also the three `rebase-push.sh --no-push` (without `--skip-if-pushed`) call sites — Step 1.m, Step 8b (issue #818), and the Rebase + Re-bump Sub-procedure — must still exist (count = 3); (I) the macro body contains the two placeholder-pinned SKIPPED format strings (`⏩ <step-prefix>: <short-name> | rebase — already pushed` and `⏩ <step-prefix>: <short-name> | rebase — already at latest main`) — these complement (D) by pinning the macro's expansion surface (D pins Verbosity Control's concrete `1.r`/`1.m` variants; I pins the macro's `<step-prefix>`/`<short-name>` placeholder variants).
